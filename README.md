# CodeCharacter Web
Web App for the frontend of CodeCharacter 2019

## Installation

```
$ npm install
```

## Running

```
$ npm start
```

## Build

```
$ npm run build
```

## Format with Prettier & Run ESLint

```
$ npm run lint
```

## Run ESLint Fix

```
$ npm run lint:fix
```

## Run Jest Tests

```
$ npm test
```
